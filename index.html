CSS display property

All the different options
Name of the option What it does My comments
none element not displayed no space allocated to it on the page
inline displays an element as an inline element height an width have no effect, margin/pading not respected
block starts on a new line , and takes up all the width
inline-block allows to set width and height of element, does not add a line break after an element
content Makes the container disappear, making the child elements children of the element the next level up in the DOM
flex makes that element a flex container
